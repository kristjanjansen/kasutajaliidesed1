### About

Interactive slides for UX course in Tartu University CS department

**Live slides:** http://kristjanjansen.ee/kasutajaliidesed1

### Installation

``` bash
git clone https://github.com/kristjanjansen/kasutajaliidesed1
cd kasutajaliidesed1
yarn
npm run dev
```

### Production

```bash
npm install -g now
now
```
